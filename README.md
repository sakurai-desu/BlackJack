## 必要なもの

1.ランダムにカードを配る(2 マイ) 2.カードの合計表示 3.カードを引くかの判定 4.ディーラーが引くかの判定 5.勝敗の確定

## 実装画面

ディーラーに「ランダムで選ばれた数値」が配られました
あなたに「ランダムで選ばれた数値」が配られました
ディーラーに「ランダムで選ばれた数値」が配られました

## 特殊な仕様

A は１と 11 両方に使える
現在の数値が 10 以下の時は、11 として扱う
それ以外の時は 1 にする
